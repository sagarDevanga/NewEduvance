1) Please download node

2) You will need to do this
a) npm install -g react-native-cli
b) npm install -g create-react-native-app

3) Please download Android Studio

4) Update the platforms using SDK MANAGER. 

5) SET Path : F:\Sdk\platform-tools // location to your platformTools

6) SET ANDROID_HOME : F:\Sdk // location to your SDK

For default android settings the SDK location would be 
C:\Users\**username**\AppData\Local\Android\android-sdk

8) To run the app -> cd into the folder where you can see node_modules and then run the below command.
npx react-native run-android
